[개인 프로젝트] 숫자야구

자바스크립트를 이용하여 숫자야구 게임을 구현한 개인 프로젝트입니다.

* 기능 설명
- 좌측에 있는 도움말 / How To Play 버튼을 눌러 게임 설명을 확인할 수 있습니다. 버튼을 다시 누르면 도움말 창을 닫을 수 있습니다.
- 중앙 상단에 있는 3개의 원에 숫자를 하나씩 입력하여 세자리 정수를 입력할 수 있습니다.
- 다음과 같은 경우 오류 메세지를 출력하고 채점하지 않습니다.
1) 한 칸 이상이 비어있는 경우
2) 숫자가 아닌 다른 문자가 포함되어 있는 경우
3) 첫자리에 0이 들어간 경우
4) 같은 숫자가 2개 이상 포함된 경우
5) 이미 정답을 맞춘 경우
- 매 채점마다 alert 메세지로 결과를 알려주고, 아래 'YOUR TRY' 란에 결과들이 하나씩 출력됩니다.
- 정답을 맞추면 alert 메세지로 알려주고, 숫자 칸이 초록색으로 바뀝니다.

* 게임 설명
- 무작위로 정해지는 세 자리 정수를 추측하는 게임입니다. 정수는 아래 조건을 만족합니다.
1) 0으로 시작하지 않는다.
2) 중복되는 숫자는 없다.
- 세자리 정수를 입력하면 채점 결과가 나오며, 결과는 아래와 같은 의미를 가집니다.
^ n Strike: n개의 숫자는 정답에 포함되며 위치까지 맞다.
^ n Ball: n개의 숫자는 정답에 포함되지만 위치는 다르다.
^ Out: 입력한 모든 숫자가 정답에 포함되지 않는다.
- 채점 결과들을 종합하여 정답을 추리하세요!
